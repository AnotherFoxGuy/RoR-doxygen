Detailed Description

Date
4th of January 2009
Author
Thomas Fischer This is a global configuration hub. Values from both config file and command line are propagated here and converted to GVars - see 'ParseGlobalVarSetting()' for details. Entries without corresponding GVar can be read by ad-hoc by macros like SSETTING(), BSETTING() etc...
